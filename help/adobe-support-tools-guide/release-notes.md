La nouvelle expérience de création de dossiers présente un formulaire actualisé conçu pour simplifier la résolution des problèmes et comprend :

![Correctif](../adobe-support-tools-guide/assets/fix.svg) - Correctifs et améliorations
![Bug](../adobe-support-tools-guide/assets/bug.svg) - Problèmes connus

## Version 1.0 – Formulaire de création de dossier amélioré

*vendredi 15 janvier 2026*

![Nouveau](../adobe-support-tools-guide/assets/new.svg) Le formulaire de cas est organisé en un flux guidé, aidant les utilisateurs à comprendre quelles informations sont requises à chaque étape :

- [!UICONTROL Sélection de produits]
- [!UICONTROL Détails du problème]
- [!UICONTROL Informations système]
- [!UICONTROL Impact]
- [!UICONTROL Coordonnées]
- [!UICONTROL Réviser et soumettre]

![Nouveau](../adobe-support-tools-guide/assets/new.svg) champ **[!UICONTROL Nouvelles étapes à reproduire]** pour capturer des détails exploitables et accélérer le dépannage.

![](../adobe-support-tools-guide/assets/new.svg)Ajout **de nouveaux champs[!UICONTROL &#x200B; de contexte] d’environnement supplémentaires** pour les produits autorisés afin de capturer les détails critiques :

- **Marketo**
   - ID Munchkin
- **Adobe Target**
   - Nom de l’activité
   - URL du site (nom de propriété des balises)/journal de certification ou de certification
- **Adobe Analytics**
   - RSID (en anglais seulement)
   - URL du site (nom de propriété des balises)/HAR ou journal de certification/cURL/journal de débogage
   - Workspace lien court
- **Adobe Journey Optimizer (AJO)**
   - ID ou URL de parcours
   - Exemple de profil
- **Platform de données client en temps réel (RTCDP)**
   - ID du composant concerné (ID de destination, ID de profil, ID d’audience, ID de jeu de données ou ID de flux de données)
   - Fichier HAR / Journaux d’assurance
- **Customer Journey Analytics (CJA)**
   - Workspace projet
   - Nom de propriété des balises


![Nouveau](../adobe-support-tools-guide/assets/new.svg) Ajout d’un **panneau[!UICONTROL &#x200B; de Recommendations piloté par]** l’IA pour afficher des conseils utiles sans interrompre le flux de création des cas.

![Nouveau](../adobe-support-tools-guide/assets/new.svg) Ajout d’une [!UICONTROL étape Résumé] des révisions pour fournir une vue consolidée de toutes les informations saisies et permettre aux utilisateurs de :

- Examiner les détails du dossier en un seul endroit
- Revenir aux étapes précédentes pour apporter des modifications
- Retour au résumé sans perdre la progression

![Correction](../adobe-support-tools-guide/assets/fix.svg) du champ de description de cas renommé en *[!UICONTROL « Veuillez décrire le problème »]* pour plus de clarté.

![Correction](../adobe-support-tools-guide/assets/fix.svg) de l’ajout d’astérisque (*) en tant qu’indicateurs de champ obligatoires pour garantir l’exhaustivité et réduire les erreurs de soumission.
